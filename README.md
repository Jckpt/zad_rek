# zad_rek

## Inicjalizacja

Aby zainicjalizować projekt, wykonaj następujące kroki:

1. Sklonuj repozytorium:

   ```sh
   git clone https://github.com/Jckpt/zad_rek
   ```

2. Przejdź do katalogu projektu:

   ```sh
   cd zad_rek
   ```

3. Zainstaluj zależności:

   ```sh
   npm install
   ```

4. Uruchom projekt:
   ```sh
   npx ts-node index.ts
   ```
